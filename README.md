# API Wrapper (PokeApi v2)

Simplified interaction with [PokeApi](https://pokeapi.co/) with selected endpoints.

## Ruby Version

- Ruby 3.0.2p107

## Rails Version

- Rails 7.0.4.2

## Installation

## **ENDPOINTS**

- ### **Pokemon**
  How to Query?
  > http[]()://localhost:3000/api/pokemon/{name-of-pokemon}
